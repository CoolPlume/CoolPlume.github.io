AI 的使用方式

官网直接访问

这是最简单粗暴的方式——直接打开 AI 工具的官网即可开始体验。 以 ChatGPT 的使用为例：

  1. 进入官网：

https://chatgpt.com/

  • 注意：需要科学上网（翻墙）才能访问。参考 《100岁老奶奶看了都能学会的“翻墙”教程》
  1. 注册并登录账号
  • 注册完成后，就可以直接使用 ChatGPT 免费套餐。
  • 如果需要更高级的模型或需要更多的使用次数，则需要购买相应的套餐
  • 不同套餐的差异可见：

https://openai.com/zh-Hans-CN/chatgpt/pricing/

ChatGPT 充值方法：

方法一：境外付款方式直充

  • 使用 Visa 或 MasterCard（需支持境外支付）
  • 使用 GooglePay 等境外支付软件

方法二： Apple 账号充值

  1. 准备一台 iPhone / ipad / Mac ，以及一个美区的 Apple ID
  2. 在美区的 AppStore 下载 ChatGPT 应用
  3. 登录你自己的 ChatGPT 账号
  4. 为你的美区 Apple 账号充值 20 美元
  • 方法一：通过境外的 Visa / MasterCard 直充
  • 方法二：通过购买充值卡充值（例如在淘宝、闲鱼等平台）
  • 例如：支付宝搜索 Pockyt Shop ，购买 Apple Gift Card US
  1. 在 Apple 设备上的 ChatGPT 应用中购买套餐

此外，不同地区的 ChatGPT 的价格会有所不同，例如尼日利亚区的价格为 9900 尼日利亚奈拉，折合约 6.+ 美元

API 访问

什么是 API ？

  • API（应用程序编程接口）：可以理解成后端提供的一个“调用入口”。 你只需要按照它的规则，把问题（参数）丢给它，它就会把答案（结果）返回给你。至于后端内部是怎么处理的，你完全不用操心
  • 举个例子：
    • 我们从 ChatGPT 的 API 官网申请到一段 API Key，通过适当的解析，就可以在别的地方调用 ChatGPT
    • 现在很多翻译软件和编程工具都内置了对多家 AI API 的支持，只要在设置中填写 API Key 和相关参数，就能直接从普通翻译切换成 AI 翻译
    • 也有一些小网站一次性集成了多家 AI 的 API，进入网站后选择想要的模型就能使用。不过这种方式由于不是在官网使用，一些功能会缺失，例如无法跨对话记忆，有些甚至不支持文件或图片传输

官网直接使用和 API 的区别

对比项 官网 API
使用方式 打开官网即可使用 申请 API Key，通过支持 API 的软件或代码调用
功能 功能完整，可跨对话记忆、支持文件 / 图片 功能可能受限，通常只支持单次对话的上下文
计费方式 按月开套餐 按 Token 数量计费

API 的计费方式

  • API 是按 Token 数量计费的
  • Token 不是字数，但可以近似理解成字的片段，例如 “你好，世界！” 这句话大约是 6 个 Token
  • 每次调用会分别计算输入和输出的 Token 数量
  • 不同模型价格不同，例如（参考 ChatGPT-4o）
    • 输入：1M Token ≈ 2.5 美元
    • 输出：1M Token ≈ 10 美元
    • 带推理功能、越高级越新的模型价格会更高
  • 需要注意的是，API 调用时上下文会随着对话增加而变长，因此输入的 Token 会越来越多，费用也会随之增加

小网站的使用

  • 以我正在使用的小网站为例：
  1. 打开网址（含我的推广码）

https://chatapi.nloli.xyz/register?aff=53nA

  1. 完成注册并登录账号
  2. 点击侧边栏“钱包”，充值一定金额
  3. 点击侧边栏“令牌管理”，随后点击“添加令牌”
    • 随意设置一个令牌名称
    • 勾选“无限额度”
    • 点击“提交”
  4. 点击“聊天”按钮旁的小箭头，点击“Lobe Chat”
  5. 在新打开的页面中，点击聊天框的第一个按钮，即可选择模型
  6. 享受您的 AI ！

提示：无论哪种方式，请根据所在地区法律法规合理使用 AI 工具
